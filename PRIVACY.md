# SafePulse Privacy Policy

Effective date: May 29, 2026

SafePulse is built by Jojin John. It is a local Android DNS/VPN protection app that helps block known ad, tracker, consent, redirect, and risky-domain requests.

## What SafePulse Collects

SafePulse does not require an account and does not collect personal profile information.

The app stores the following data locally on the device:

- protection status and settings
- blocklist update metadata
- allowlist and per-app bypass choices
- blocked/checked/allowed request counters
- recent blocked domains and local diagnostic logs

These values are used only to run protection and show the dashboard.

## VPN Permission

SafePulse uses Android's `VpnService` permission to receive DNS requests locally on the device. This is required for DNS-level blocking.

SafePulse does not operate a remote VPN server. DNS filtering happens locally, and allowed DNS queries are forwarded to the DNS provider selected in the app.

## Network Requests

SafePulse may contact:

- configured blocklist source URLs to download fresh rules
- selected upstream DNS providers to resolve allowed DNS requests
- GitHub Releases to check for app updates

## What SafePulse Does Not Do

SafePulse does not sell personal data.

SafePulse does not upload browsing history to a SafePulse server.

SafePulse does not include advertising SDKs.

SafePulse does not require login, signup, or user tracking accounts.

## User Control

Users can stop protection at any time from inside the app. Users can clear stats and logs from the Protection screen.
